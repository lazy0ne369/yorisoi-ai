# Yorisoi AI (寄り添いAI)

Connecting care, AI that stays by your side.

## Overview

Yorisoi AI is a multi-agent eldercare platform designed to address Japan's aging population and caregiver shortages through intelligent health monitoring, safety assessment, care coordination, and AI-driven orchestration.

## Agents

- Health & Wellness Agent
- Safety & Emergency Agent
- Care Coordination Agent
- AI Care Manager

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- Groq API
- Supabase
- Vercel
