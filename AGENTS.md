# Yorisoi AI (寄り添いAI)

Tagline:
Connecting care, AI that stays by your side.

## Project Overview

Yorisoi AI is a multi-agent eldercare platform designed to address Japan's aging population challenges.

The platform coordinates multiple AI agents to assist elderly individuals, families, caregivers, and care coordinators.

## AI Agents

### Agent 1 - Health & Wellness Agent

Responsibilities:

- Health monitoring
- Medication adherence tracking
- Risk prediction
- Wellness recommendations

### Agent 2 - Safety & Emergency Agent

Responsibilities:

- Fall detection
- Emergency assessment
- Safety monitoring
- Escalation recommendations

### Agent 3 - Care Coordination Agent

Responsibilities:

- Caregiver assignment
- Scheduling
- Family notifications
- Care plan execution

### Agent 4 - AI Care Manager

Responsibilities:

- Multi-agent orchestration
- Risk prioritization
- Unified care planning
- Executive summaries

### Agent 5 - UI/UX Architect

Responsibilities:

- Dashboard architecture
- Design system
- Frontend architecture
- User experience

## Design Principles

- Human-centered
- Accessible
- Explainable AI
- Elder-friendly
- Japanese-inspired
- Mobile responsive

## Coding Standards

- TypeScript everywhere
- Use App Router
- Use Server Components when possible
- Keep components small and reusable
- Use shadcn/ui before creating custom components
- Prefer composition over duplication

## Project Goal

Demonstrate a multi-agent AI system that improves elderly care coordination, safety, and family awareness through intelligent orchestration.

## Language Requirements

The application UI must be fully English.

Allowed Japanese:

- Project branding
- Person names

Not allowed:

- Mixed-language interfaces
- Japanese labels
- Japanese navigation
- Japanese alerts
- Japanese dashboard text

Assume judges are international.

All generated content should be English-first.
